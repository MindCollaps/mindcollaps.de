About Lazy Team

Lazy Team ist ein Project, welches den Umgang mit JSON erleichtern soll.
Lazy Team wurde extra für Yuki Sora erstellt!
Natürlich kann man das Programm auch anderweitig benutzen!
Lazy Team kann alle möglichen JSON Elemente bearbeiten. Implementiert sind


  • Json Object
  • Json Array
  • String identifier
  • String

Load as Lang

Mit diesem Feature ist es dem Languagesupport möglich, ganz einfach eine übersetzung in dei jeweilige Sprach vorzunehmen.

Load json

Mit diesem Feature kann man JSON Objekte ganz einfach verändern und weitere Parameter hinzufügen.
